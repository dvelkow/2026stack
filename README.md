# 2026 Stack Readout

This repo contains a single interactive HTML readout for my 2026 supplement stack experiment.

I am running this as a personal experiment to find the most optimal supplement stack for my own situation: a lean 24-year-old who works intensive hours behind a screen while staying active. The goal is not to build the biggest stack possible, but the highest ROI stack: the best price-to-effect ratio for cognition, sleep, aesthetics, hormonal support, resilience, and long-term health.

The HTML file roughly illustrates:

- which compounds contribute to each target area
- the estimated direction and size of the effects
- how long a person on the stack might take to feel or stabilize those effects
- the difference between same-day effects and slower saturation effects over weeks

## Stack Notes

Core compounds:

- Creatine
- Citicoline
- Omega-3
- D3 + K2
- Astaxanthin
- Zinc
- Magnesium L-threonate
- Glycine
- Copper
- Semax

Bonus compounds:

- NAC
- L-theanine
- Boron

NAC, L-theanine, and boron are included because they may add useful upside in specific areas, but they are less central to the stack than the core compounds above.

This is a personal tracking and visualization project, not medical advice. The estimates in the HTML are directional and experimental, not clinical claims. Anyone considering supplements should do their own research and talk to a qualified medical professional when appropriate.
